
Packages and package managers
-----------------------------
In linux package is nothing but a bundle of files archived into a single file. these packages will contain everything 
which is required to install and run the application.


application packages:
--------------------

.deb  --> Debian, Ubuntu, Linux mint
.rpm  --> RHEL, centos, amzon linux, fedora

package managers:
----------------
apt --> ubuntu, debian
yum/dnf --> RHEL, centos, Amazon Linux
apk  --> alpine Linux

apt install nginx
yum install nginx


dpkg -i filename.deb

systemctl status <application-name>
systemctl restart <application>
systemctl stop <application>
systemctl enable <application>  -->to restart/start application by default whenever there is a reboot of server
systemctl disable <application> --> to not automatically start on system reboot

journalctl -u <application>  --> to see the application status logs
 sudo apt update --> to update with the versions in your respository
 
 sudo apt upgrade --> to upgrade already installed applications / install latest versions
 sudo apt remove <application>  --> removes/uninstall an application
 sudo apt purge <application>  -> completely removes application and its configuration files
 sudo apt autoremove <application> --> it will remove dependency softwares from your system
 
 
zip:
---
to archive single/multiple files or directories into one package file.zip

zip <filename.zip> file1 file2 directory1 directory2


zip -r <filename.zip> direcroty/

unzip:
------
to extract a zip package
unzip -l <zip filename> --> to see the contents of the archive

unzip <zip filename> --> extracts in the current working direcotry

unzip <zip filename> -d <destination path>  --> to store the extracted contents in this destination path

tar:
----
tar will archive and alos compresses the archieve
it will keep the directory/file permissions

.tar --> just archive
.tar.gz --> archieve and compresses


tar -cvf <file.tar> direcotry/ file1 file2

tar -xvf <file.tar>   ---> extract to current working direcotry

tar -tvf <file.tar>  ---> list the contents

tar -cvzf <file.tar.gz> directory/ filename   ---> creates a archive 
tar -xvzf <file.tar.gz> -C /opt  --> extracts in that directory path
