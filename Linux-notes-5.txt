File/Directory Permissions:
---------------------------

user(u) --> who has created this file. who is the owner of this file
group(g) --> group also want to access this file
others(o) --> everyone else

3 types of permissions

read (r=4)
write (w=2)
execute(x=1)

r-x
r-x

rwxrwxr--

file:
-----
read(r=4) -> whoever has read access can see the contents in the file
write(w=2) --> he can edit this file
execute(x=1) --> he can execute a binary or shell script


directory:
---------
read(r=4) --> list the contents inside the direcotry
write(w=2) --> enter (cd) into the direcotry
execute(x=1) --> create, delete, rename files inside the direcotry

chmod: change mode
------------------
used to change permission for a file or directory

chmod [options] mode file/directory

user - rwx  7 
group- rw   6
others- r    4

chmod 764 test.txt
chmod u+rwx,g+rw,o+r test.txt
chmod a+x 
chmod a-x 

chmod u-x test.txt


0 -> no permission --> ---
1 -> execute --> --x
2 -> write --> -w-
3 -> write and execute --> -wx
4 -> read --> r--
5 -> read and execute --> r-x
6 --> read and write  --> rw- 
7 --> all permission--> rwx


chmod -R 777 directory
chmod -v 664  file/directory


chown: change owner
-------------------
used to change the user or group ownership of a file/directory

chown [options] user[:group] file/directory

chown user filename
chown :group filename
chown user:group file

chown -R user:group  directory
