Direcoty Structure:
-------------------

/ --> root directory
/bin --> contians ready-to-use programs/ binaries/executables. normal users have access
/sbin --> cotains system binaries. only root user have access to these commands to execute.
/dev --> devices files 
/home/ubuntu --> holds the personal directories for regular users
/root  ---> home directory for root user
/etc  --> the core system configuration directory
/proc --> contains information about the current running processes and some kernel parameters like cpu and memory information
/run --> contains runtime data stored in this directory for system purpose which will erased for reebot
/opt --> used to install third party softwares.
/sys --> same as /proc. contians more information system and interface
/tmp --> temporary. every user will have all the access to this directory. 
/usr --> user system reources or binaries will be here
/boot --> booting related information
/var --> variable information related directory will be under this var directory
